## Dev mode

Install

``
npm install
``

You need .env variables:

```
VITE_API_BASE_URL_OPENWEATHER=
VITE_APP_ID_OPENWEATHER=
VITE_API_BASE_URL_MAPBOX=
VITE_MAPBOX_PK=
```

Run

``
npm run dev
``

Open: [localhost:5173](http://localhost:5173/)
